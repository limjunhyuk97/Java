# Polymorphism(다형성)

## Polymorphism이란?
  - **하나의 요소(객체)** 가 **하나 이상의 타입** 을 갖는 것이다.
  - 다형성의 결과는, **하나의 메시지**에 대한 **(하나의 요소/객체 -> 하나 이상의 타입)다양한 대답의 가능성**이다.
    - **하나의 객체 - 하나이상의 타입**
    - **하나의 메시지 - 하나이상의 메소드**
    - **같은 역할, 같은 책임 - 다른 타입의 객체**
